# Azhunholding MMC - Rəsmi Saytı

Bu layihə Xankəndi şəhərində yerləşən Azhunholding MMC un məmulatları zavodu üçün hazırlanmış vizitkart tipli veb-saytdır.

## Şirkət Haqqında
Azhunholding MMC Azərbaycan Respublikasının Xankəndi şəhərində un məmulatları hazırlayan böyük bir zavoda malikdir. Şirkət Qarabağ regionunda hərbi hissələri, mağazaları, dövlət orqanlarını və özəl şirkətləri keyfiyyətli çörək məhsulları ilə təmin edir.

## Texnologiyalar
- HTML5
- Tailwind CSS (CDN vasitəsilə)
- JavaScript (Vanila JS)
- Font Awesome (İkonlar üçün)

## Yerləşdirmə (Deployment)
Bu sayt Firebase Hosting və ya Hostinger vasitəsilə yerləşdirilmək üçün hazırdır.

### Firebase Hosting:
1. `firebase login`
2. `firebase init`
3. `firebase deploy`
